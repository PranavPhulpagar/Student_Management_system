# Student_Management_system
A simple Java-based Student Management System that allows users to add, view, and delete student records using a console interface. The project uses file handling to store data permanently and demonstrates core Java concepts such as OOP, collections, and exception handling.


## Features
- Add new student with ID, name, and age
- View all students
- Search student by ID or name
- Delete student by ID
- Data persistence using a text file (`students.txt`)
- Prevents duplicate student IDs
- Easy, menu-driven interface

## Technologies Used
- Core Java
- File I/O (`File`, `Scanner`, `PrintWriter`)
- Object-Oriented Programming with `ArrayList`
- Exception handling

## How to Run
1. Clone or download this repository.
2. Open a terminal/command prompt in the project directory.
3. Compile the Java file:
   ```bash
   javac StudentManagementSystem.java
